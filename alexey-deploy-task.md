**Что это**

На сайте lux-floor.de появится кнопка-ассистент. Посетитель нажимает, вводит вопрос или запрос, ассистент задаёт уточняющие вопросы, подбирает пол из каталога и создаёт лид в Bitrix24. Всё работает автоматически. Тебе нужно один раз развернуть сервер и вставить одну строку в сайт.

|  |
| --- |
| **Что Олег уже сделал:** написал весь код, проверил на живом каталоге, подготовил конфигурацию сервера. Тебе ничего настраивать не нужно, только нажать несколько кнопок и вставить ключи. |

**Шаги**

**Шаг 1. Создать аккаунт на Render (2 минуты)**

* render.com
* Нажать "Get Started for free"
* Зарегистрироваться через GitHub (или email)
* Подтвердить email если просит

*Render — это сервер-хостинг, на котором будет жить ассистент. Бесплатная регистрация.*

**Шаг 2. Создать новый веб-сервис (3 минуты)**

* В дашборде нажать "New" > "Web Service"
* "Connect a repository" > вставить ссылку на репозиторий (Олег пришлёт)
* Render сам найдёт все настройки из файла в репо. Ничего не менять.
* "Create Web Service"

*Render автоматически читает файл render.yaml из репозитория и знает как собрать и запустить сервис.*

**Шаг 3. Ввести секретные ключи (5 минут)**

В настройках сервиса: Environment > Environment Variables. Добавить 4 ключа. Значения Олег пришлёт отдельным сообщением (не в этом файле).

|  |  |
| --- | --- |
| **Название ключа** | **Что это** |
| ANTHROPIC\_API\_KEY | Ключ Claude AI (мозг ассистента) |
| WOO\_CONSUMER\_KEY | Ключ WooCommerce (доступ к каталогу) |
| WOO\_CONSUMER\_SECRET | Секрет WooCommerce |
| BITRIX\_WEBHOOK\_URL | Вебхук Bitrix24 (ты уже его создавал) |

|  |
| --- |
| **Важно:** ключи вводить точно как написано, без пробелов до и после знака =. Не пересылать их в открытых чатах. |

**Шаг 4. Перейти на Starter план - $7 в месяц (2 минуты)**

* Billing > Upgrade to Starter
* Ввести карту (Ильи или рабочую)

*На бесплатном плане сервер засыпает через 15 минут без посетителей. Первый посетитель после сна ждёт 30 секунд. Starter ($7/мес) держит сервер живым постоянно.*

**Шаг 5. Скопировать URL и отправить Олегу (1 минута)**

* После деплоя Render покажет URL сервиса
* Вид: find-your-floor.onrender.com (или похожий)
* Скопировать и отправить Олегу

*Олег проверит что ассистент отвечает, и пришлёт готовую строку для вставки в сайт.*

**Шаг 6. Вставить код в WordPress (5 минут) - после подтверждения от Олега**

После того как Олег подтвердит, что ассистент работает:

* Вариант A (проще): установить плагин

WordPress > Plugins > Add New > искать "Insert Headers and Footers" (автор WPBeginner) > Install > Activate.

* Вариант B (без плагина):

Appearance > Theme Editor > footer.php

В обоих случаях вставить одну строку кода перед тегом </body>. Строку пришлёт Олег после шага 5.

|  |
| --- |
| <script src="https://find-your-floor.onrender.com/widget.js"  data-backend="https://find-your-floor.onrender.com"></script> (точный URL Олег подтвердит) |

После вставки: открыть lux-floor.de в браузере, убедиться что в правом нижнем углу появилась кнопка ассистента.
